# Social Media Analytics Dashboard

An interactive Power BI dashboard that analyzes social media usage patterns and their relationship with users' mental well-being.

##  Objectives

- Analyze participant demographics.
- Understand social media usage behavior.
- Explore the relationship between digital habits and mental well-being.
- Present insights through an interactive dashboard.

---

## Key Insights

- Users who use social media more frequently at night tend to sleep fewer hours.
- Using social media more freqently is assosiated with higher anxiety levels.
- Higher notification frequency is associated with higher FOMO scores.
- Increased social comparison is associated with lower self-esteem.
- Users who attempted a digital detox reported a slightly higher life satisfaction.
